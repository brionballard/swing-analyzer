# Sample swings

Drop a short swing clip here (e.g. `swing1.mp4`) and push it to the branch to
have it analyzed. Video files in this folder are allowed by git (see the
`.gitignore` exception); keep clips short (2–6s, whole body in frame).

For each clip, note the **view** (face-on / down-the-line) and **handedness**
(right / left) so the metrics are computed correctly.
